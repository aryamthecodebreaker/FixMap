# Code of Conduct

FixMap is committed to a welcoming, useful, and harassment-free community.

## Expected behavior

- Be respectful, specific, and constructive.
- Critique ideas and code rather than people.
- Assume good intent while still naming concrete impact.
- Respect privacy, repository licenses, and security-reporting boundaries.
- Help newcomers understand the project instead of dismissing incomplete questions.

## Unacceptable behavior

Harassment, threats, discrimination, sexualized attention, deliberate intimidation, doxxing, and sustained disruption are not accepted in project spaces.

## Reporting

For community conduct concerns, contact the maintainer privately through the email address on the maintainer's GitHub profile. For product vulnerabilities, use the private process in [SECURITY.md](SECURITY.md).

Maintainers may remove content or restrict participation when behavior conflicts with this policy. Reports will be handled as privately and fairly as practical.
