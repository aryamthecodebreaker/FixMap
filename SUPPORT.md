# Support

## Questions and ideas

Use [GitHub Discussions](https://github.com/aryamthecodebreaker/FixMap/discussions) for usage questions, workflow ideas, and examples you want to share.

## Bugs

Open a [bug report](https://github.com/aryamthecodebreaker/FixMap/issues/new?template=bug_report.yml) with the command, expected behavior, output, and a minimal repository shape when possible. Remove secrets and proprietary source before posting.

## Security

Do not open a public issue for a vulnerability. Follow [SECURITY.md](SECURITY.md).
